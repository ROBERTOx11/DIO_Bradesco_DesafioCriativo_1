Desafio Criativo: Extraindo Insights do Feedback de Clientes Bancários
O obejetivo desta atividade é desenvolver a habilidade de criar um prompt capaz de extrair insights do feedback de clientes bancários.
O desafio é concluído após o cumprimento de todas as etapas sugeridas pelo professor/mestre organizador da atividade
